StepZen - Frankfurter GraphQL API

Frankfurter is an open-source API for current and historical foreign exchange rates published by the European Central Bank. Using GraphQL! View GraphiQL playground

frankfurter_historical_rates

Response

Returns a Frankfurter_Historical_Rates

Arguments
Name Description
amount - Float
date - Date
from - String
to - String

Example

Query
query frankfurter_historical_rates(
  $amount: Float,
  $date: Date,
  $from: String,
  $to: String
) {
  frankfurter_historical_rates(
    amount: $amount,
    date: $date,
    from: $from,
    to: $to
  ) {
    amount
    base
    date
    rates
  }
}
Variables
{
  "amount": 123.45,
  "date": "2007-12-03",
  "from": "xyz789",
  "to": "xyz789"
}
Response
{
  "data": {
    "frankfurter_historical_rates": {
      "amount": 987.65,
      "base": "xyz789",
      "date": "2007-12-03",
      "rates": {}
    }
  }
}

Types

Boolean

Description

The Boolean scalar type represents true or false.

Types

Date

Description

Date scalar type represents a date. Date is serialized as an RFC 3339 section 5.6 full-date quoted string

Example
"2007-12-03"
Types

Float

Description

The Float scalar type represents signed double-precision fractional values as specified by IEEE 754.

Example
123.45
